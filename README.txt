餐廳 POS V2.1.6 重製完整版

本版整合重點：
1. 商品編輯加入商用等級驗證
2. 商品名稱未輸入不可新增 / 保存
3. 商品金額未輸入、非數字、或 <= 0 不可新增 / 保存
4. 顯示紅框與錯誤提示，保存按鈕依合法性自動鎖定
5. 商品列表內可直接修改價格
6. 商品列表內可直接展開編輯口味模組
7. 新增商品按鈕移入商品編輯區
8. 模組必選改為開關式，摘要同步更新
9. 點餐頁版面不變
10. 購物車相同商品 + 相同口味 + 相同備註自動疊加
11. 檔案結構維持 pages / modules / core / styles 分離

部署方式：
- 直接整包覆蓋到 repo 根目錄
- 重新部署 GitHub Pages
- 若是 PWA，請重新整理或重新加入主畫面
